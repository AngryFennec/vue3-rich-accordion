<ul>

    <li>SSR friendly</li>
    <li>Block resize friendly</li>
    <li>v-model controlled</li>
    <li>Default-open feature</li>
    <li>Dynamic content reaction</li>
    <li>Dynamic items allowed</li>

    <li>Css fully customizable</li>
    <li>W3C validator passed</li>
